Welcome to the Grafana Automation App

This application automates the creation of Grafana datasources and dashboards.

Architecture Diagram

|   Frontend      |----->|   Backend       |----->|   Grafana API   |
| (HTML/CSS/JS)   |      |  (Node.js)      |      |                 |
+-----------------+      +-----------------+      +-----------------+

Architecture Narrative

The application is a simple web app with a Node.js backend and a plain HTML, CSS, and JavaScript frontend. The backend acts as a proxy to the Grafana API, handling authentication and forwarding requests. The frontend provides a user-friendly interface for creating datasources and dashboards.

Workflow Narrative

The user selects the desired action (create datasource or dashboard) from the menu. They then fill out the required information and submit the form. The frontend sends a request to the backend, which then makes the appropriate API call to Grafana. The response from the Grafana API is then displayed to the user.
